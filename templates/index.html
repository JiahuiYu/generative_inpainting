Generative Image Inpainting

: No worries! AI can remove unwanted objects from your photo.

GitHub Repo    Open API on Ainize

Note

  • Results are direct outputs from trained generative neural networks. No post-processing steps are applied.
  • Model is trained on CelebA-HQ and Places2 datasets.

How to use

  1. Select an option between Places2 and CelebA-HQ.
  2. Insert an image.
  3. Paint a portion you want to remove using the white brush.
  4. Click run button and get the result.

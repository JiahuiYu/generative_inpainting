Image Inpainting

: No worries! AI can remove unwanted objects from your photo.
With this AI open-source project, you can remove unwanted object of photo, you can eraser person, era

GitHub Repo    Open API on Ainize

Note

  • Results are direct outputs from trained generative neural networks. No post-processing steps are applied.
  • Model is trained on CelebA-HQ and Places2.

How to use

  1. Select a model between Places2 and Celeba.
  2. Insert an image.
  3. Mask the part you want to erase white.
  4. Click run button and get the result.
